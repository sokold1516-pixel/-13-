**Выполнено:**
- ✅ Добавлено поле `ImageData` типа `VARBINARY(MAX)` в таблицу `Exhibitss`
- ✅ Все ~100 записей заполнены тестовыми бинарными данными
- ✅ Обновлён скрипт инициализации `database.sql`
- ✅ Протестировано чтение/запись BLOB-данных

- Заполнение: массовый `UPDATE` с тестовым PNG-плейсхолдером
- Совместимость: Microsoft SQL Server 2016+
